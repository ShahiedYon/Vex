\# Vex (Vextly Automation System)



Vex is an automation engine built to:

\- Generate leads (Debt, Solar, Automation)

\- Execute tasks via scripts

\- Manage queue-based workflows

\- Run browser, Python, and Node tasks



\## Structure



\- scripts/ → execution logic

\- tasks/ → task queue system

\- memory/ → persistent AI identity

\- config/ → rules + scoring

\- data/ → external datasets



\## Run Example



powershell -ExecutionPolicy Bypass -File scripts/phase14\_run.ps1
